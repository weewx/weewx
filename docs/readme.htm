weewx: Documentation Guide

Quick Start guides:

  • Quick Start Guide for Debian Linux. For Debian, Ubuntu, Mint, Wheezy, using DEB packaging.
  • Quick Start Guide for Redhat Linux. For Redhat, CentOS, Fedora, using RPM packaging.
  • Quick Start Guide using setup.py. For all operating systems. This is the traditional weewx install. You should use this approach if you plan on doing extensive customization.
User's Guide
Contains complete details about how to install, configure, and run weewx. Includes instructions for migrating from wview. This is the definitive guide to using weewx.
Customization Guide
Describes how to modify reports through configuration options as well and templates. Explains how to create and customize skins. Also includes more advanced topics such as creating custom generators and services, customizing the internal engine, and creating drivers for new weather station types.
Upgrade Guide
Instructions for upgrading from previous versions of weewx.
Change Log
List of what has changed in each version.
SheevaPlug
Notes about installing on a SheevaPlug.
